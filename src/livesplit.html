LiveSplit

LiveSplit Fork

Fork of LiveSplit featuring quality of life changes.

  • Distributed comparison similar to that of splitmaker.
  • Add option to create a comparison for the finished run if saving the splits when it is not a Perosnal Best.
  • "Silver Segments", a way of changing the color of the split/segment if it ties the gold for the segment.
  • Worst Split Times comparison - self explanatory.
  • Random Image option for layout background. Select a folder of images and anytime LiveSplit restarts a new image will be selected.
  • "Copy Splits" button that turns the Personal Best comparison into text.
  • Option to prevent LiveSplit from resizing via the mouse.
